## Arithmetic Operator

**Hackerrank Problem:** [Arithmetic Operations](https://www.hackerrank.com/challenges/bash-tutorials---arithmetic-operations/problem)

#### How to run
```
./Solution.sh
```

#### INPUT EXAMPLES
```
5+50*3/20 + (19*2)/7
-105+50*3/20 + (19^2)/7
(-105.5*7+50*3)/20 + (19^2)/7
```

#### OUTPUTS FOR ABOVE EXAMPLES
```
17.929
-45.929
22.146
```
